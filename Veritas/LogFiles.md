Info about some log files in /etc/vx/log
```
•	cmdlog                 records Veritas commands that have been run and the time and date when they were run 
•	ddl.log                  records changes to DMP nodes internally within Veritas kernel
•	dmpevents.log records DMP events such as path enabled or disabled
•	translog                Veritas is transaction oriented. This log file contains details of each atomic transaction that takes place on VX metadata
```
